N4语法：たら、と、ば、なら的辨析完整汇总

发表于2023-07-28|更新于2023-07-29|N4语法
|阅读量:
たら、と、ば、なら是初学日语中较难的一个知识点，在下面的文章中，我会归纳出它们的用法并且加以详细的解释

と 的用法 😸

恒定条件

表示条件的情况下，有一种“前句是后句发生的条件”一样的感觉，前项发生，后项也会发生，表示的是习惯与往复。多用于描述自然现象、天然现象、真理、伦理、道理、事实、生理现象等。

  • 暖かくなると 、桜の花が咲きます
    天气转暖，樱花盛开 (出自《日本语表现文型》)
  • この道を右に曲がると 、駅が見えます
    在这条路右转就可以看到车站 (出自《日本语表现文型》)
  • ボタンを押すと 、 電気がつく
    按下按钮，灯就会亮
  • 1に２をたすと 、３になる
    1加2等于3

当我们使用と的时候，尽量会使用现在时去表述事实，不会使用た型（否则会有一种“个例”的感觉）

表示习惯

如果是表述特定的个人行为，例如人的性情和习惯的一般行为，可以用と

  • 私はおなかがすくと 、いつもラーメンを作って食べます
    我肚子饿了，就煮拉面吃 (出自《日本语表现文型》 )
  • 兄はお酒を飲むと 、顔が赤くなります
    哥哥喝了酒就会脸红 (出自《蓝宝书》N4部分语法 )
  • 彼は１ちょっとからかわれると 、すぐに本気で怒りだす
    别人一取笑他，他就真的动怒了

注意，と的后句往往是不带意志的，是恒常性的动作，所以表示“劝诱、命令、意志”的句子是不能作为后句的 !
比如下面的句子：
週末になると、公園にいきましょう ❌
眠いと、寝なさい ❌

过去发生的事情/意外发现

表示“一…就（发现）…”，表示动作之后立即出现的意外发现

  • 目を覚ますと 、病院のベッドの上だった
    当我醒过来，人已经躺在医院病床上了

表示动作的连续(てすぐ)

表示“一做完（某事）就（做另一件事）”，表示动作之后紧接着另一个动作

  • 兄は上着を着ると 、黙って出ていきました
    哥哥穿好上衣，一声不吭就出去了 (出自《日本语表现文型》 )
  • このごろ、マリはうちに帰ってくると 、すぐどこかへ電話を掛けます
    最近玛莉一回家，就会打电话 (出自《日本语表现文型》 )

固定用法:「Vないと」

表示催促、提醒他人或者自己要采取的行动

  • -A : 会議の時間が変わったことを木村さんにも知らせないと
    -B : そうですね
    -A : 得告诉木村会议时间的变动
    -B : 是啊
    (出自《日本语表现文型》)
  • ともちゃん、早く起きないと ．もう七時半ですよ
    Tomo酱，该起来了，已经七点半了啊 (出自《日本语表现文型》)

たら的用法 💹

表示“之后”（その後で）

表示“…之后”，前句中的动作必然会发生但没有开始，一旦开始就要做后句，后句可以使用意志、想法、意见、劝导等的句子

  • 京都駅に着いたら 、私に電話をください．すぐ迎えに行きます
    到了京都站请给我打个电话，我马上就会过去接你的 (出自《日本语表现文型》)
  • 今日の仕事が終わったら 、飲みに行こう
    忙完今天的工作，咱们就去喝一杯吧 (出自《蓝宝书》N4语法 )
  • あと、帰ったら 、洗濯物を取り込んでおいてね
    回去之后再把衣服收一下哦 (出自《浪客剑心 明治剑客浪漫谭（新作）》)

如果

表示“如果，倘若”，假设前句的事情万一发生的话，后句的事情就发生。常与“もし”连用。可以使用意志、想法、意见、劝导等的句子

  • もし、おもしろい本があったら 、買ってきてください
    有好看的书的话，还请你买回来 (出自《日本语表现文型》)
  • もし明日は雨だったら 、運動会は中止になります
    明天要是下雨，运动会将会取消 (出自《蓝宝书》N4语法 )

意外或者出乎意料的发现

前面的事情之后，遇到了后面的事情。一般前面的事情不管发不发生，后面的照样且必然发生，也就是说前后之间没有必然的关系。但也有前后互相作用的情况。后面的事情多为发生的意外或者出乎意料的发现。

  • お正月に実家に帰って、引き出しの中を整理したら 、高校時代のアルバムが出てきた
    正月回到老家，整理了一下抽屉，翻出了高中时代的相册 (出自《蓝宝书》N4语法)
  • 先週の日曜日にデパートへ行ったら 、人でいっぱいだった
    上个星期天去百货商店，那里人特别多

一般来说，这种用法和と近似，二者都可以使用

  • 山に登ったら、都市の全景が見えた ✔️
  • 山に登ると、都市の全景が見えた ✔️

固定用法：たらどう

类似于たほうがいい，但是语句更加直接。除此以外我们还可以用たらいかが、たら来代替

  • A: バイト、そんなに大変なら、辞めたらどう ?
    B: やめられないんですよ
    A: 打工这么苦，还是辞掉了吧
    B: 我辞不了啊
    (出自《日本语表现文型》)

ば的用法 🎱

如果ば前面跟着的是ナ形容词或者名词，可以用であれば（可以理解成である的假定）或者なら・ならば（可以理解成だ的假定）

恒定条件

表示必然的恒定条件，例如揭示事物真理、自然现象、事实等，这里和と类似。隐含的语义就是，必须满足A，才能得到B，B一般来说，多为积极的语句

  • ２に２をたせば 、４になる （ ２に２をたすと、４になる 🆗）
    ２加2等于４
  • 冬になれば 、雪が降る （ 冬になると、雪が降る 🆗）
    到了冬天就会下雪

表示假定

相当于“…的话”、“如果….的话”，后面多为表示意志、愿望、希望、建议等的语句，和たら差不多

  • 欲しければ 、持って行けよ
    你想要的话就拿走吧
  • 明日、天気が良ければ テニスをしますが、よくなければ うちでDVDでもみます
    明天要是天气好，就打网球，不好的话就在家里看DVD

注意！这种用法只适用于前句是状态动词、名词或者形容词的情况，例如「あれば」「いれば」「欲しければ」。但如果是非状态动词，后面就不能跟着意志、请求的句子！

例如《日本语表现文型》中的一个例子：
もし熱が出れば、この薬を飲んでください ❌
もし熱が出たら、この薬を飲んでください ✔️

なら的用法 🐰

表示假定

和たら、ば类似，都表示假定，后面可以接上表示意志、愿望、希望、建议等的语句

  • 明日雨が止むなら 、遠足に行こう
    明天要是雨停了，就去郊游吧
  • そんなにほしいなら 、持っていきなさい
    你要是那么想要的话，就拿去

表示条件

表示的条件可能是前→后，但也有后句的事件发生在前句的事件之前的情况，这一点是与其它三个词最明显的不同！请看下面的句子——

  • この本を読むなら 、貸してあげるよ
    你要读这本书的话，我借你
    👉 借完之后，再看书
  • 乗るなら 、飲むな
    开车不喝酒
    👉 喝完酒，再开车

其实たら也可以表示先做后面的，再做前面的动作，把たら改成んだったら即可

总结 🐇

在B站up主的一个视频中，详细地对比了这四个词的用法，其中列举的表格，可以带大家参考且理解

结果 假定 确实未来 助言命令
と 🔴 ❌ ❌ ❌
ば 🔺 🔴 ❌ 🔺
たら 🔴 🔴 🔴 🔴
なら ❌ 🔴 ❌ 🔴

结果 : 后面是前面发生的事情的结果（例如自然现象、习惯以及发生意外的事）
假定 : 前面的不知道会不会发生
确实未来 : 未来100%会发生的事情 （上文提到的第一个たら的用法，肯定会来东京站之类的）
助言命令 ：基于前面的事情，后面给对方建议或者命令（ば的局限性见上文）

一言以蔽之，たら几乎是万能的!

练习 💌

下面来看看几个例子

例子1

  • 春になると、桜の花が咲く。
  • 春になれば、桜の花が咲く。
  • 春になったら、桜の花が咲く。
  • 春になるなら、桜の花が咲く。

上述的句子，哪些对，哪些错？

  • 春になると、桜の花が咲く。 ✔️
  • 春になれば、桜の花が咲く。 ✔️
  • 春になったら、桜の花が咲く。 ❓
  • 春になるなら、桜の花が咲く。 ❓

①-②句完全通顺（自然变化），但是③-④有些别扭

例子2

  • 新しいPCを買うと、電話してね。
  • 新しいPCを買えば、電話してね。
  • 新しいPCを買ったら、電話してね。
  • 新しいPCを買うなら、電話してね。

上述的句子，哪些对，哪些错？

  • 新しいPCを買うと、電話してね。 ❌
  • 新しいPCを買えば、電話してね。 ❌
  • 新しいPCを買ったら、電話してね。 ✔️
  • 新しいPCを買うなら、電話してね。 ✔️

第一句，と后面不能接上意志、劝诫、建议等语句
第二句，ば后面如果表示意志、建议等，前面不能接上非状态的动词
第三句和最后一句明显正确

拓展：辨析たらいい/といい/ばいい

  1. 如果用在表示愿望的句子中，都可以互换。后面多跟随「なあ/のに/けど/が」等词语
  • 雨が降ったらいいなあ ○
  • 雨が降るといいなあ ○
  • 雨が降ればいいなあ ○
    要是下雨就好了

注意啦！一般「といい」是不和带有意志的动词在一起的，要和非意志动词在一起
(意志动词就是人为动作之类的动词，非意志动词不随着人的意志为转移，多为自然现象等，此外，可能型也是非意志动词)
【错误用例】
テストに合格するといい
这是错误的句子，我们可以改成
テストに合格できるといい

  1. 多用在建议的句子中，注意：除了「といい」，其余二者都可以用在疑问的用法中
  • 疲れているようですね、今仕事も忙しくないから、2、3日休んだらいい ですよ
    看样子你累了吧，今天工作也不忙，你先休息两三天吧 (出自《日语表现文型》)
  • どうしたら・すればいい の?
    该怎么办才好?
