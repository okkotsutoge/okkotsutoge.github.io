たら、と、ば、なら的辨析

Created2023-07-28|Updated2023-07-28|语言学习
|Post View:

たら、と、ば、なら是日语中比较难以辨别的四个用法，在网站中，有一幅图概括了大致的区别：

概括起来，就是

と：必然　　ば：原則　　たら：実現　　なら：仮定

と 的用法

表示条件关系

表示条件的情况下，有一种“前句是后句发生的条件”一样的感觉，前项发生，后项也会发生。例如自然现象、机械操作

  • 暖かくなると 、桜の花が咲きます
    天气转暖，樱花盛开 (出自《日本语表现文型》)
  • この道を右に曲がると 、駅が見えます
    在这条路右转就可以看到车站 (出自《日本语表现文型》)
  • 兄はお酒を飲むと 、顔が赤くなります
    哥哥喝了酒就会脸红 (出自《蓝宝书》N4部分语法 )
  • ボタンを押すと 、 電気がつく
    按下按钮，灯就会亮

表示习惯

表示不带有太多个人意志的习惯，恒常性的动作

  • 私はおなかがすくと 、いつもラーメンを作って食べます
    我肚子饿了，就煮拉面吃 (出自《日本语表现文型》 )

注意，と的后句往往是不带意志的，是恒常性的动作，所以表示“劝诱、命令、意志”的句子是不能作为后句的 !
比如下面的句子：
週末になると、公園にいきましょう (×)

过去发生的事情/意外发现

  • 目を覚ますと 、病院のベッドの上だった
    当我醒过来，人已经躺在医院病床上了

表示动作的连续(てすぐ)

表示“一做完（某事）就（做另一件事）”，表示动作之后紧接着另一个动作

  • 兄は上着を着ると 、黙って出ていきました
    哥哥穿好上衣，一声不吭就出去了 (出自《日本语表现文型》 )
  • このごろ、マリはうちに帰ってくると 、すぐどこかへ電話を掛けます
    最近玛莉一回家，就给某处打电话 (出自《日本语表现文型》 )

第二个例句的と后句是现在时，并且是最近常有的动作，也可以理解成是习惯做的、经常做的事。

固定用法:「Vないと」

表示催促、提醒他人或者自己要采取的行动

  • -A : 会議の時間が変わったことを木村さんにも知らせないと
    -B : そうですね
    -A : 得告诉木村会议时间的变动
    -B : 是啊
    (出自《日本语表现文型》)
  • ともちゃん、早く起きないと ．もう七時半ですよ
    Tomo酱，该起来了，已经七点了 (出自《日本语表现文型》)

たら的用法

表示“之后”（その後で）

表示“…之后”，后句一般就是做某事或者成为XXX，并且后句可以使用意志、想法、意见、劝导等的句子

  • 京都駅に着いたら 、私に電話をください．すぐ迎えに行きます
    到了京都站就给我打个电话，我会赶紧过来的 (出自《日本语表现文型》)
  • 今日の仕事が終わったら 、飲みに行こう
    忙完今天的工作就去喝一把吧 (出自《蓝宝书》N4语法 )

如果

表示“如果，倘若”，常与“もし”连用

  • もし、おもしろい本があったら 、買ってきてください
    有好看的书的话，还请你买回来 (出自《日本语表现文型》)
  • もし明日は雨だったら 、運動会は中止になります
    明天要是下雨，运动会将会取消 (出自《蓝宝书》N4语法 )
