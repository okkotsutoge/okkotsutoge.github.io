方法 说明
all() 查询所有记录，返回QuerySet查询对象
get() 查询符合条件的单一记录
filter() 查询符合条件的多条记录
exclude() 查询符合条件之外的全部记录

上面我们介绍的仅仅是等值的查询，如果是非等值的查询，则需要Django的谓词查询来进行实现。

💡 组成：属性+__（双下划线）+特定的词（gte，lt等）

上面介绍的等值匹配，谓词是_exact

模糊查询

谓词_contsians查询包含特定的值的内容，例如语句Student.objects.filter(name_contains='Ja')，相当于SQL语句的

💡 除此以外，我们还有其它可用的谓词：
  • _startswith：以XXX开始

_gt 大于
_gte 大于等于
