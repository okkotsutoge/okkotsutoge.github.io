尊敬语 💟

尊敬语的场合

表示说话人对话题人或者听话人给予高级礼遇，对话题人或听话人言行、性质、状态机器所属事务表示尊敬使用的方式。表达敬意的对象一般为身份高的、年龄老者、生疏者、客人等上位者。

常用名词

人称代词

日文 翻译
あなた 您
この方 这位
そのかた 那位
あのかた 那位
こちらさま 这位
閣下(かっか) 阁下，您
貴兄(きけい) 兄台
父上(ちちうえ) 父亲

人名、职称后面跟着的表达：さん、先生、殿(どの)、さま。「さん」多用于会话中，「殿」接在名词、职称后面，仅用于公务性文件和信函。在信函中，多用「様」，对于自己的老师可以用「先生」，公文抬头要用「殿」

带有「お・ご」的名词

一般来说，「お」用在日语的固有词之前，「ご」用在日语的音读词之前，但也有例外。

  • 以「お」开头的词：お所、お湯、お鍋、お皿、お箸
  • 以「ご」开头的词：ご住所、ご意見、ご職業、ご希望、ご都合、ご自身
  • 例外：お仕事、お電話、お時間、お洋服、お料理、お弁当、お誕生日

不会带有「お・ご」的名词有：、

  1. 日语里习惯上基本不会用外来语带上「お・ご」
  2. 音节越多的词，越不会带「お・ご」
  3. 首音节为「お」的词，不会使用「お」
  4. 表示自然现象、矿产矿物、工业机械、团体组织的，一般不带「お・ご」，例如お雪、お学校、ご会社都是不正确的

带有数量的名词

  1. ご一人（一位）
  2. お一人/ お二人（仅限1、2人）
  3. お一方（ひとかた）/お二方（ふたかた）/お三人方（さんにんかた）（只限称谓1-3人）
  4. お一人さん/ お二人さん/ お三人さん （一位、两位、三位，商业气息很浓）
  5. 一名様（めいさま）/二名様/三名様…

句型

「お・ご」~になります

变形规则 💰

  1. 与动词的连用型一起使用，使用「お…になります」

    原型 「お・ご」~になります
    かきます おかきになります
    読みます お読みになります
    話します お話しになります
    出ます 出になります

  2. 与汉字词一起使用，使用「ご…になります」

    原型 「お・ご」~になります
    出席します ご出席になります
    利用します ご利用になります
    予約します ご予約になります

注意⚠️
使用该句型时有以下四个要点值得注意：
（1）此语法 不能与单音节词根动词组合，如：「みる」「する」「くる」
（2）此语法 不能与特殊的敬语动词结合，如：「くれる」「行く」「言う」
（3）此语法 不能与外来语结合，如：「パソコン」「ダイエット」「コーヒー」
（4）此语法 不能与特定的某些词汇结合，如：「失敗」「営業」「運転」

例句💱

  • 山田さんはお戻りになりました か
    山田先生回去了吗
  • どうぞ、おかけになってください
    请坐
  • 今からは社長がお話しになります
    现在由社长发言
  • ご乗車になる 方は、お急ぎください
    上车的乘客请抓紧时间
  • ご出席の場合は早めにお知らせください
    出席的地点请尽早通知
  1. 「お・ご…になってください」可以换成「お・ご…ください」，表示请求别人做某事，如上句的第四句的「お急ぎください」、第五句的「お知らせください」
  2. 「お・ご…になる方」可以换成 「お・ご…の方」，例如第四句可以改成「ご乗車の方」

なさる

变形规则 💰
お／ご＋サ変動詞語幹 ＋なさる / なさい

解释 🍜
这种形式的尊敬较有时代感，并且更加有正式的庄重感。该句型可以使普通动词尊敬语化，但仅限于サ变动词「する」。「食べる」「歩く」等动词的变化形不能后接「なさる」。

例句 ⏺️

  • 先生は先ほどご帰宅なさいました
    老师刚刚回家去了
  • こちらにご記入なさってください
    请在此处填写

れる／られる

变形规则 💰
一类动词：い段→あ段 + れます
二类动词：去掉ます + られます
三类动词：する→される 来る→来られる

「おる」变成「おられる」的形式一般不怎么用到，平时最常用到的是 いらっしゃる

讲解 🍜
敬语助动词「れる」「られる」，接在动词未然形的后面构成敬语动词，表示对于动作主体的尊敬。其中五段动词未然形后接「れる」，一段和カ变动词未然形后接「られる」，サ变动词词干后接「される」。尊敬的程度相对较轻，在工作中经常见到。但是由于它与被动语态的变形几乎重合，所以容易造成误解，或者与固定用法冲突，失去其尊敬的语气。

例句 💥

  • 課長は明日会社に来られますか
    明天课长会来公司吗
  • 先生はこの本を読まれました
    老师读了这本书

お・ご…です

和「お・ご…になる」一样，表示对他人的尊敬，经常有「..ている」的语感，当然，表示过去和将来的例子也有存在。

  • 部長はもうお出かけです か
    部长已经出去了么「部長はもう出かけていますか」
  • 友達をお待ちです か
    在等朋友吗「友達を待っていますか」
  • いつ、ご出発です か
    几点出发呢

お・ご…になれる

接续 📅
同「お・ご…になる」

解释 🍄
表示”您能够…、您可以…”

例句 🚝

  • 市民図書館は祝日でも通常とおりにご利用になります
    市民图书馆在节假日照常开放
  • インターネットでもお電話でもお申し込みになれます
    上网和打电话都可申请
  • 入場券はオンラインでもお求めになれます
    入场券可以网上线上购买
  • このカードは日本国内のみでご利用になれます
    这张卡只能在日本国内使用
  • ここのWi-Fiは無料でお使いになれます
    这里的WiFi可以免费使用
  • 料理はお持ち帰りになれます
    料理可以打包带走
